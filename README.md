# Boo Profile Preview

This repository contains a local preview of a Boo profile saved using the SingleFile browser extension.

## Files

- `index.html` - main page that embeds `mySavedBooProfile.html` in an iframe.
- `mySavedBooProfile.html` - saved Boo profile content.
- `myBooProfile.jpeg` - image asset for the profile.

## How to preview locally

1. Open a terminal in `d:\Boo\m`
2. Run:
   ```powershell
   python -m http.server 8000
   ```
3. Open your browser to:
   - `http://localhost:8000/`
   - or `http://localhost:8000/index.html`

## Editing and testing changes

- Edit `index.html` and/or `mySavedBooProfile.html`.
- Save the files.
- Refresh the browser page.
- If the page still looks old, use a hard refresh: `Ctrl+F5`.

## Notes

- `mySavedBooProfile.html` is generated by the SingleFile plugin. If it looks broken in GitHub Pages, re-save it with SingleFile and then refresh locally.
- The local server automatically serves the latest saved files, so you do not need to restart it after editing HTML/CSS.

## GitHub Pages testing

This repo uses a single GitHub Pages URL: `https://gawhary.github.io/m/`.

If you want a separate preview environment for a work-in-progress branch, use one of these options:

- Test locally with the command above.
- Create a separate test repo and deploy that branch there.
- Use a custom GitHub Actions workflow to publish WIP output under a subpath such as `/wip/`.
